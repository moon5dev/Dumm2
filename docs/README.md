# inventory-viewer
A prototype project for building a simple WMS inventory search screen.

This project does not aim to build a full WMS.  
The goal is to implement a single inventory viewer screen with a practical structure similar to real business applications.

## Goals
- Build an inventory search screen with Ext JS
- Implement a Spring Boot API
- Use MyBatis XML Mapper
- Call a stored procedure through MyBatis
- Understand the basic inventory lookup domain
- Keep useful examples for MyBatis, procedure calls, and logging configuration(for real usage)

## Scope
### Included
- Search by item code
- Search by item name
- Search by warehouse/plant
- Search by location
- Search by stock status
- Display inventory list in a grid
- Show selected row details

## Tech Stack
### Frontend
- Ext JS

### Backend
- Java 17
- Spring Boot
- MyBatis
- Stored Procedure

### Database
- SQL Server

### Build
